計算公式與推導過程

1. 蒸發損失 (Evaporation, E)

E = 循環水量(公制) × 溫差(ΔT) × 0.00153

計算：{{ formatNum(L_base) }} × {{ formatNum(deltaT_base) }} × 0.00153 = {{ formatNum(E_base) }} m³/hr ↳ 換算：{{ formatNum(E_base * 4.40287) }} GPM ｜ {{ formatNum(E_base * 0.27778) }} L/s

2. 飛濺損失 (Drift, W)

W = 循環水量(公制) × 飛濺率(%)

計算：{{ formatNum(L_base) }} × {{ W_pct }}% = {{ formatNum(W_base) }} m³/hr ↳ 換算：{{ formatNum(W_base * 4.40287) }} GPM ｜ {{ formatNum(W_base * 0.27778) }} L/s

3. 排放損失 (Blowdown, B)

B = E / (COC - 1) - W

計算：{{ formatNum(E_base) }} / ({{ COC }} - 1) - {{ formatNum(W_base) }} = {{ formatNum(B_base) }} m³/hr ↳ 換算：{{ formatNum(B_base * 4.40287) }} GPM ｜ {{ formatNum(B_base * 0.27778) }} L/s
