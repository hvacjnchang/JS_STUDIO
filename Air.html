附錄：計算公式與過程 (Calculation Details)

1. 基礎熱力學公式

  • 飽和蒸氣壓 Pws = 0.61078 × exp(17.27 × T / (T + 237.3)) kPa
  • 濕度比 W = 0.621945 × Pw / (Patm - Pw) kg/kg
  • 焓值 h = 1.006 × T + W × (2501 + 1.86 × T) kJ/kg
  • 比容 v = 0.287042 × (T + 273.15) / (Patm - Pw) m³/kg

2. 室內負荷計算

質量流率 (m) = 風量 / 比容
= {{ printDetails.room.flow_m3s.toFixed(4) }} / {{ printDetails.room.v.toFixed(4) }} = {{ printDetails.room.m.toFixed(4) }} kg/s

焓差 (Δh) = (RSH + RLH) / m
= ({{ printDetails.room.rsh_kW.toFixed(2) }} + {{ printDetails.room.rlh_kW.toFixed(2) }}) / {{ printDetails.room.m.toFixed(4) }} = {{ printDetails.room.dh.toFixed(2) }} kJ/kg

濕度比差 (ΔW) = RLH / (m × 2501)
= {{ printDetails.room.rlh_kW.toFixed(2) }} / ({{ printDetails.room.m.toFixed(4) }} × 2501) = {{ printDetails.room.dw.toFixed(5) }} kg/kg

3. 混合點計算

質量比例 A (mA) = {{ printDetails.mix.flowA }} / {{ printDetails.mix.vA.toFixed(4) }} = {{ printDetails.mix.mA.toFixed(2) }}

質量比例 B (mB) = {{ printDetails.mix.flowB }} / {{ printDetails.mix.vB.toFixed(4) }} = {{ printDetails.mix.mB.toFixed(2) }}

混合焓值 (hmix) = (mAhA + mBhB) / (mA + mB)
= ({{ printDetails.mix.mA.toFixed(2) }} × {{ printDetails.mix.hA.toFixed(2) }} + {{ printDetails.mix.mB.toFixed(2) }} × {{ printDetails.mix.hB.toFixed(2) }}) / {{ (printDetails.mix.mA + printDetails.mix.mB).toFixed(2) }} = {{ printDetails.mix.hMix.toFixed(2) }} kJ/kg

混合濕度比 (Wmix) = (mAWA + mBWB) / (mA + mB)
= {{ printDetails.mix.wMix.toFixed(2) }} g/kg

4. 盤管冷卻能力

質量流率 (m) = 風量 / 比容
= {{ printDetails.coil.flow_m3s.toFixed(4) }} / {{ printDetails.coil.v.toFixed(4) }} = {{ printDetails.coil.m.toFixed(4) }} kg/s

冷卻能力 (Q) = m × |hA - hB|
= {{ printDetails.coil.m.toFixed(4) }} × |{{ printDetails.coil.hA.toFixed(2) }} - {{ printDetails.coil.hB.toFixed(2) }}| = {{ printDetails.coil.q.toFixed(2) }} kW

5. 恆溫恆濕計算

質量流率 (m) = {{ printDetails.th.flow_m3s.toFixed(4) }} / {{ printDetails.th.v.toFixed(4) }} = {{ printDetails.th.m.toFixed(4) }} kg/s

純溫差顯熱 (Qsensible) = m × (1.006 + 1.86 × WA) × |TB - TA|
= {{ printDetails.th.m.toFixed(4) }} × (1.006 + 1.86 × {{ printDetails.th.wA.toFixed(5) }}) × |{{ printDetails.th.tB.toFixed(2) }} - {{ printDetails.th.tA.toFixed(2) }}| = {{ printDetails.th.q_sensible.toFixed(2) }} kW

加濕水量 (mw) = m × (WB - WA) × 3600
= {{ printDetails.th.m.toFixed(4) }} × ({{ printDetails.th.wB.toFixed(5) }} - {{ printDetails.th.wA.toFixed(5) }}) × 3600 = {{ printDetails.th.mw.toFixed(2) }} kg/h

供水焓值 (hwater) = 4.186 × {{ printDetails.th.waterTemp }}°C = {{ printDetails.th.h_water.toFixed(2) }} kJ/kg

水滴蒸發吸收熱量 (Qevap) = (mw / 3600) × (2501 + 1.86 × TB - hwater)
= ({{ printDetails.th.mw.toFixed(2) }} / 3600) × (2501 + 1.86 × {{ printDetails.th.tB.toFixed(2) }} - {{ printDetails.th.h_water.toFixed(2) }}) = {{ printDetails.th.q_evap.toFixed(2) }} kW

蒸汽焓值 (hsteam) = 2676 kJ/kg

加濕器熱負荷 (Phum) = (mw / 3600) × (hsteam - hwater)
= ({{ printDetails.th.mw.toFixed(2) }} / 3600) × (2676 - {{ printDetails.th.h_water.toFixed(2) }}) = {{ printDetails.th.p_hum.toFixed(2) }} kW

除濕水量 (mw) = m × (WA - WB) × 3600
= {{ printDetails.th.m.toFixed(4) }} × ({{ printDetails.th.wA.toFixed(5) }} - {{ printDetails.th.wB.toFixed(5) }}) × 3600 = {{ printDetails.th.mw.toFixed(2) }} kg/h

凝結潛熱 (Qlatent) = (mw / 3600) × (2501 + 1.86 × TB)
= ({{ printDetails.th.mw.toFixed(2) }} / 3600) × (2501 + 1.86 × {{ printDetails.th.tB.toFixed(2) }}) = {{ printDetails.th.q_latent.toFixed(2) }} kW

凝結水帶走熱量 (Qcond) = (mw / 3600) × (4.186 × 10°C) = {{ printDetails.th.q_cond.toFixed(2) }} kW
